# Spread Folder Overview (English)

This folder contains the **public‑facing materials** of the UMBL project, including official PDF documents in both English and Chinese, as well as their editable Markdown source files.

UMBL (Universal Meaning‑Based Language) is an early‑stage **pictographic language‑design development project**.  
It explores how a structured, compositional, and evolvable pictographic writing system might be designed.

The purpose of the Spread folder is to provide **ready‑to‑use materials** for instructors, students, researchers, and the general public.

## 📌 Why both PDF and Markdown?

- **PDF**: polished, shareable, suitable for teaching, presentations, and public distribution  
- **Markdown**: editable, version‑controlled, ideal for GitHub and collaborative updates  

Markdown serves as the “source,” while PDF serves as the “published version.”

---

## 📘 What’s included in Spread?

- **Project overview (CN/EN)**  
- **Student thesis topic guide (CN/EN)**  
- **Research topic list (CN/EN)**  
- **Promo copy for public platforms (CN/EN)**  

All materials reflect UMBL’s identity as an **open, experimental, research‑friendly language‑design prototype**.

---

## 📮 How to use this folder

- To quickly understand the project → read README_CN.md / README_EN.md  
- To introduce UMBL to others → use the promo materials (PDF or md)  
- For academic work → refer to the thesis guide and topic list  
- To update content → edit the Markdown files in the `md/` directory  

If you need to revise or expand any document, please modify the Markdown version first and export a new PDF.
