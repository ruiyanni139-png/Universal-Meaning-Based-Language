# Spread 文件夹说明（中文）

本文件夹用于存放 UMBL 项目的**对外传播材料**，包括中英文 PDF 正式版，以及对应的 Markdown 源文件。  
UMBL（Universal Meaning‑Based Language）是一个处于早期探索阶段的**象形语言设计开发项目**，旨在研究如何构建一种结构化、可组合、可演进的象形文字语言原型。

Spread 文件夹的作用是：  
**为教师、学生、研究者与公众提供可直接使用的介绍材料与宣传内容。**

---

## 📌 为什么同时保留 PDF 与 Markdown？

- **PDF**：适合正式传播、展示、投稿、打印  
- **Markdown**：适合 GitHub 在线阅读、版本控制、协作编辑  

Markdown 是“源文件”，PDF 是“发布版本”。

## 📮 使用方式

- 想快速了解项目 → 阅读 README_CN.md / README_EN.md  
- 想向他人介绍项目 → 使用宣传文案（PDF 或 md）  
- 想做学术研究 → 阅读选题指南与选题清单  
- 想编辑内容 → 使用 md 文件夹中的 Markdown 源文件  

如需更新内容，请直接修改 md 文件，再导出 PDF。
