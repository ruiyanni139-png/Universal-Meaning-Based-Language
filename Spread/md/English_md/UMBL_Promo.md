# 1. Standard Version (Medium  Reddit)

UMBL (Universal Meaning‑Based Language) is an ongoing pictographic language‑design project.  
It is not an icon set, not a UI library, and not a branding system.  
UMBL is an attempt to explore what a structured, compositional, evolvable pictographic language could look like if we designed one from scratch.

The project grew out of long‑term observations comparing Chinese and English

- Chinese builds meaning through character formation, radicals, and visual structure  
- English builds abstraction through roots, affixes, and morphology  
- Chinese media culture leans heavily toward visual‑semantic expression (subtitles, danmu overlays)  
- Phonetic writing systems face challenges in cross‑cultural comprehension  
- Natural languages are not always consistent in logic, abstraction, or semantic transparency  

These observations led to a simple question

If we could redesign a language, could it be both visually intuitive and structurally systematic

UMBL does not claim superiority over existing languages.  
It is a language engineering prototype, created to explore

- How to define pictographic morphemes  
- How to combine components into words  
- How to construct pictographic sentences  
- How to design grammar, formation rules, and evolution mechanisms  

UMBL is still in an early stage, but it already serves as

- A research object for linguistics, cognitive science, design, and computer science  
- A source of undergraduate thesis topics  
- A platform for discussing how languages can be engineered, not just inherited  

If you’re interested in conlangs, visual communication, cross‑cultural semantics, or language design as an engineering discipline, UMBL might be worth following.
